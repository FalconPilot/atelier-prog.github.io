Un petit serveur Web

Apprendre à créer un serveur Web (HTTP) minimaliste

Dans cet atelier, on vous propose de créer un petit serveur web. Il n'est pas nécéssaire d'être trop ambitieux, un simple serveur que l'on peut lancer affichant les pages web qu'on lui demande est largement suffisant.
