Un petit serveur Web

Apprendre à créer un serveur Web (HTTP)

Dans cet atelier, on vous propose de créer un petit serveur web. Il n'est pas nécéssaire d'être trop ambitieux, se limiter à GET et POST est largement suffisant. (Biensur vous pouvez faire plusieurs propositions dont certaines ne gérant pas GET ou POST).
L'idée de l'atelier est de comprendre le fonctionnement général d'un serveur web. Rien de plus.
