Un livre d'or comme un webservice

Premier webservice exposé

Être capable d'interagir avec le monde extérieur est une chose, exposer une interface pour interagir avec sa propre application en est une autre. Dans cet exercice, on vous propose de créer un livre d'or sous forme de webservice. Exposant au minimum ces fonctionnalités :

  • Récupérer toute la liste de messages en JSON
  • Ajouter un message au livre d'or
  • Récupérer la liste de X messages
En plus de l'exposition du webservice, vous pouvez réaliser une application d'exemple. On vous conseille de vous documenter sur "REST" pour rédiger cette application.

Pour les systèmes un peu exotique, n'hésitez pas à rédiger un README pour expliquer comment "builder" et "lancer" votre petite application !
