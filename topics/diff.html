Le célèbre Diff du monde Unix

Diff est une commande Unix qui permet de comparer deux fichiers et d’en afficher les différences.
Son utilisation typique consiste à calculer les changements entre une version d’un fichier et une version plus ancienne du même fichier.
Diff affiche les changements ligne par ligne pour un fichier texte.

De manière plus simple, on propose d'écrire un programme qui prend deux fichiers en argument et produit un rapport exprimant les différences entre le fichier 1 et le fichier 2. Un exemple concret de l'utilisation de diff est proposé par Github : dans le rapport de transformation lors d'un commit.
